🚀 SDDAP Projesi – Çalıştırma Talimatları

Bu proje frontend (HTML/CSS/JS) + backend (FastAPI) olarak iki bölümden oluşur.
Backend kısmı uvicorn ile çalıştırılır.
Lütfen adımları sırasıyla takip edin.

───────────────
📁 1. Ortam Hazırlığı
───────────────
1.1. Python 3.10+ yüklü olmalıdır  
    Terminale şunu yazıp kontrol edin:
    > python --version

1.2. Sanal ortam oluşturun (isteğe bağlı ama önerilir):
    > python -m venv venv

1.3. Ortamı aktifleştirin:

- Windows:
    > venv\Scripts\activate

- Mac/Linux:
    > source venv/bin/activate

───────────────
📦 2. Gerekli Paketleri Kurun
───────────────
Aşağıdaki komutla gerekli tüm kütüphaneleri yükleyin:

> pip install -r requirements.txt

───────────────
🚀 3. Backend (FastAPI) Başlatma
───────────────
Projede bir `main.py` dosyası varsa, şu komutu kullanarak sunucuyu çalıştırın:

> uvicorn main:app --reload

Eğer dosya başka dizindeyse örnek:

> uvicorn backend.main:app --reload

➡ Çalıştıktan sonra şu adresi açın:
http://localhost:8000

Dokümantasyon için:
http://localhost:8000/docs

───────────────
🌐 4. Frontend (HTML) Tarayıcıda Açma
───────────────
Frontend klasörüne gelin:

> cd frontend

Ardından tarayıcıdan direkt `index.html` veya `homepage.html` dosyasını açabilirsiniz.

Alternatif: Python ile local server başlatabilirsiniz:

> python -m http.server 5500

ve tarayıcıya girin:
http://localhost:5500

───────────────
🧪 5. Giriş Bilgisi (örnek)
───────────────
Kayıt olduktan sonra kullanıcı adı/şifreyle giriş yapabilirsiniz.
Veya kayıt endpointine manuel kullanıcı da POST edebilirsiniz.

───────────────
📎 6. Notlar
───────────────
- Backend, `auth/login` ve `auth/signup` endpointleri sağlar
- JWT token döner, frontend localStorage'da saklar
- Girişten sonra `homepage.html`'e yönlendirme olur
